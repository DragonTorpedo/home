The Models

Speed

3.5 Flash

Your default workhorse. Built for throughput — fastest latency in the lineup, ideal for any structured, repeatable task at scale.

  • Context 1M tokens
  • Output 65k tokens
  • Latency Lowest
  • Best for Extraction, Agents, Translation

Reasoning

3.5 Flash Extended

Deep reasoning via background thinking tokens. Unmatched for complex logic — but use targeted prompts to keep output grounded and concise.

  • Context 1M tokens
  • Reasoning Heavy
  • Quota cost Up to 2× faster drain
  • Best for Math, Brainstorming, Logic

Judgment

3.1 Pro

The autonomous decision-maker. Best when the path forward is unclear — nuanced reasoning, human-like tone, and recovery from unexpected failure.

  • Context 1M tokens
  • Output 32k tokens
  • Reasoning Premium
  • Best for Code, Creative, Persona

Workflow Intelligence

Taming Extended

Extended over-engineers when left unchecked — bloated formatting, artificial vocabulary. Append: Keep final output concise. Avoid excessive markdown and obscure terminology.

The Hybrid Pipeline

Use Flash to format, extract, and classify large datasets rapidly. Feed the structured result into Pro for synthesis and nuanced judgment. Never rely on one model for the full chain.

Quota Strategy

Extended burns reasoning tokens invisibly — multimodal use drains quota 2× faster. Run Flash for daily operations. Reserve Extended for complex sessions at the start of your quota week.

Context Stuffing

Stop summarizing. The 1M token window exists to hold entire repositories, draft histories, and full document sets. Upload everything once — hallucination drops, coherence holds.
